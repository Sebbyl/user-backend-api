Running instructions:
mvn clean install
mvn spring-boot:run

Hosted location:
http://localhost:8000/api/users

code location: 
src/main/java/com/boot/User/API

Post request body:
{
        "id": int id,
        "username": String username,
        "firstName": String firstName,
        "lastName": String lastName,
        "email": String email,
        "phoneNumber": String phoneNumber
    }
